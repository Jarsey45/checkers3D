# Checkers 3D

## Jak uzywać 
  #### ***Nie działa z jakiegoś powodu na Mozilla Firefox***

  Użyć komendy
  ```
  node server-node.js
  ```

## Express

  Użyć komendy
  ```
  npm install express body-parser
  ```
  Następnie
  ```
    node server-express.js
  ```
